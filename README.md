# favorite-image-collector

## Usage

```bash
export TWITTER_CONSUMER_KEY=xxxxxxxx
export TWITTER_CONSUMER_SECRET=xxxxxxxx
export TWITTER_ACCESS_TOKEN_KEY=xxxxxxxx
export TWITTER_ACCESS_TOKEN_SECRET=xxxxxxxx
```

```bash
$ yarn install
$ yarn start
```
